cdn stands for content delivery network. It is a distributed group of servers that caches web content (like images, videos, and scripts) in locations geographically close to end users

Work- if u need to load a website that has a main server hosted in latvia (fitgirl repack). Then rather than sending the request to main server, CDN intercepts the request and sent to a server closest to u.

why- It speeds up load time and reduce the server strain
